
Tyva republic wildfire and fog dataset - v4 2021-11-13 10:59am
==============================

This dataset was exported via roboflow.ai on November 13, 2021 at 7:59 AM GMT

It includes 63 images.
I-label-all-the-smoke-on-images are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)
* Resize to 640x480 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* Random brigthness adjustment of between 0 and +35 percent

The following transformations were applied to the bounding boxes of each image:
* 50% probability of horizontal flip
